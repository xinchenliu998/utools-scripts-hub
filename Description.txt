Scripts Hub 脚本管理工具

Scripts Hub 是一个基于 uTools 的脚本管理与快速启动工具，用来统一管理本地脚本，并通过关键词快速运行它们。你不需要记住脚本路径，也不需要手动打开终端或文件夹。

开源免费 https://github.com/xinchenliu998/utools-scripts-hub
---

主要功能

1. 快速运行脚本

* 在 uTools 中输入 run 即可进入运行界面
* 支持按名称、关键字、描述搜索脚本
* 使用键盘上下键选择，回车即可运行
* 脚本通过外部应用启动，不等待执行结果

2. 脚本管理

* 支持添加单个脚本文件
* 支持添加整个文件夹（包含子目录）
* 可为脚本添加描述，方便区分和搜索
* 支持编辑和删除已添加的脚本
* 支持启用/禁用脚本（禁用的脚本不会在运行界面显示）
* 支持拖拽调整脚本顺序

3. 规则配置

* 可根据文件名或后缀自动匹配运行规则
* 提供快速模板，快速配置常见文件类型规则
* 不同类型的文件可使用不同的应用打开
* 支持为应用设置启动参数
* 支持启用/禁用规则（禁用的规则不会参与匹配）
* 支持拖拽调整规则顺序（规则按顺序匹配，顺序很重要）
* 未匹配规则时使用系统默认方式打开

4. 灵活运行方式

* 所有脚本都通过外部应用运行
* 不限制脚本语言或文件类型
* 更像一个"脚本启动器"，而不是执行器

5. 个性化设置

* 支持简体中文和英文界面切换
* 自定义主题颜色
* 自定义按钮颜色（主要、危险、警告、成功、默认五种类型）
* 配置默认排除文件夹（.git、node_modules、__pycache__ 等）
* 支持重置设置到默认值

---

基本使用流程

1. 添加脚本

* 在 uTools 中输入 run-setting
* 进入设置界面，添加脚本文件或脚本文件夹
* 填写名称和描述（可选）

2. 配置规则（可选）

* 为不同文件类型指定打开方式
* 可使用快速模板快速配置常见文件类型
* 例如：

  * .js 文件用 node 打开
  * .ps1 文件用 PowerShell 打开
  * .md 文件用 VS Code 打开

3. 运行脚本

* 在 uTools 中输入 run
* 搜索脚本
* 回车或双击即可运行

4. 个性化配置（可选）

* 点击设置按钮打开设置对话框
* 在"外观"标签页切换语言和主题颜色
* 在"排除"标签页配置默认排除的文件夹
* 在"按钮"标签页自定义各类型按钮的颜色

---

适用场景

* 管理零散的本地脚本工具
* 快速启动常用脚本而不打开终端
* 统一管理不同语言、不同用途的脚本
* 把 uTools 当作脚本启动面板使用

---

特点总结

* 不依赖脚本语言
* 不限制文件类型
* 搜索快、启动快
* 规则灵活、配置简单
* 支持快速模板，配置更便捷
* 支持启用/禁用，管理更灵活
* 支持多语言界面
* 支持主题和按钮颜色自定义
* 支持文件夹排除配置
* 适合脚本较多、重度 uTools 用户
